Intel® Integrated Performance Primitives Image Blurring and Rotation Sample

The Image Blurring and Rotation sample included with the Intel® Integrated Performance Primitives (Intel® IPP) library is designed to illustrate how to use Intel IPP functions for image affine warping and filtering.

This sample application is used in the Image Blurring and Rotation with Intel® IPP tutorial document available at:

  • On Windows* OS: C:\Program files (x86)\IntelSwTools\documentation_2016\en\ipp\common\tutorials\ipp_blur_rotate\ (on some systems, instead of Program Files (x86), the directory name is Program Files)
  • On Linux* OS: /opt/intel/documentation_2016/en/ipp/common/tutorials/ipp_blur_rotate/

The tutorial provides detailed information on how to:

  • Implement box blurring with the Intel IPP filtering functions
  • Rotate an image with the Intel IPP functions for affine warping
  • Set up environment to build the sample application
  • Compile and link the application

System Requirements

  • A system based on one of the following processor types:
    • Intel® Core™ processors (2nd generation or higher recommended)
    • Intel® Pentium® processors
    • Intel® Xeon® processors
    • Subsequent IA-32 architecture based systems
    • Processors with Intel® 64 architecture
  • Intel® IPP installed as part of one of the following suites:
    • Intel® Parallel Studio XE
    • Intel® System Studio
    • Intel® Integrated Native Developer Experience (Intel® INDE)
  • OpenGL* developer package
  • X11* developer package (required only for building on Linux* OS)
